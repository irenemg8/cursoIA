Instrucciones para Utilizar los Marcos de Formulación de ChatGPT
1. Marco RTF (Role-Task-Format)

Objetivo: Utiliza este marco para definir claramente el rol, la tarea y el formato en que deseas que ChatGPT actúe o responda.
Pasos:
Role: Define el rol que debe asumir ChatGPT (ej. asistente, tutor, investigador).
Task: Especifica la tarea concreta que debe realizar (ej. responder una pregunta, explicar un concepto).
Format: Determina el formato de la respuesta deseada (ej. lista, párrafo, diálogo).
2. Marco TAG (Task-Action-Goal)

Objetivo: Este marco ayuda a clarificar la tarea, la acción requerida y el objetivo final de la interacción con ChatGPT.
Pasos:
Task: Define la tarea específica.
Action: Describe la acción que ChatGPT debe realizar para completar la tarea.
Goal: Clarifica el objetivo que se busca alcanzar con la acción.
3. Marco BAB (Before-After-Bridge)

Objetivo: Explicar un problema y su solución destacando la transformación de un estado inicial a uno final, y cómo se logra esa transición.
Pasos:
Before: Explica la situación o problema inicial.
After: Describe el estado deseado después de resolver el problema.
Bridge: Detalla los pasos o métodos para cruzar del estado "Before" al "After".
4. Marco CARE (Context-Action-Result-Example)

Objetivo: Proporcionar un contexto, describir una acción, aclarar los resultados y dar ejemplos para ilustrar el proceso.
Pasos:
Context: Ofrece el contexto o antecedentes necesarios.
Action: Describe la acción que se debe llevar a cabo.
Result: Explica los resultados obtenidos o esperados.
Example: Proporciona ejemplos concretos para ilustrar la acción y los resultados.

Cada vez que recibas un prompt o desees crear uno, sigue estos pasos para seleccionar y completar el marco más adecuado:

Paso 1: Identificar el Propósito del Prompt

Antes de elegir un marco, determina qué quieres lograr con tu interacción con ChatGPT. ¿Estás buscando una explicación, una solución a un problema, una acción específica o un ejemplo ilustrativo?
Paso 2: Elegir el Marco Adecuado

Basado en tu objetivo, elige uno de los siguientes marcos:
RTF (Role-Task-Format): Si necesitas que ChatGPT asuma un rol específico y respondas en un formato particular.
TAG (Task-Action-Goal): Si tu foco está en realizar una tarea específica con un objetivo claro.
BAB (Before-After-Bridge): Si quieres explicar un cambio o transformación debido a una solución o intervención.
CARE (Context-Action-Result-Example): Si necesitas contextualizar una acción y mostrar resultados con ejemplos.
Paso 3: Formular el Prompt con Detalles

Una vez seleccionado el marco, haz preguntas específicas para llenar cada componente del marco elegido:
Para RTF:
¿Qué rol debe asumir ChatGPT? (Ej. entrenador, consultor)
¿Cuál es la tarea específica? (Ej. enseñar un concepto, proporcionar consejos)
¿En qué formato debería entregarse la respuesta? (Ej. lista, narrativa, diálogo)
Para TAG:
¿Cuál es la tarea exacta?
¿Qué acción debe realizar ChatGPT?
¿Cuál es el objetivo final de esta acción?
Para BAB:
¿Cómo es la situación antes de la intervención?
¿Cómo mejora o cambia después?
¿Cuáles son los pasos específicos que forman el puente entre antes y después?
Para CARE:
¿Cuál es el contexto o la situación inicial?
¿Qué acción se va a describir?
¿Cuáles son los resultados esperados o logrados?
¿Puedes dar un ejemplo concreto que ilustre estos puntos?
Paso 4: Implementar el Prompt

Utiliza las respuestas a las preguntas para formular un prompt claro y estructurado. Asegúrate de que todas las partes del marco estén completas y bien definidas.
Paso 5: Evaluar y Ajustar

Después de recibir la respuesta de ChatGPT, evalúa si cumplió con el objetivo esperado. Si es necesario, ajusta el prompt y prueba diferentes configuraciones del marco para optimizar los resultados.
